# Фитнес-бот для ВКонтакте

Персональный помощник для тренировок с генерацией программ, питанием, статистикой и напоминаниями.

## Возможности
- Регистрация с выбором цели, места, сплита
- Программа тренировок на месяц (Excel + текст)
- Питание по целям
- Отметка тренировок + статистика
- Календарь тренировок (в разработке)
- Достижения (бейджи)
- Напоминания о тренировках (APScheduler + часовые пояса)
- Настройки профиля

## Стек
- Python 3.10+
- vk-api
- openpyxl
- APScheduler
- pydantic-settings
- SQLite

## Запуск
1. Склонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте файл `.env` с `VK_TOKEN` и `GROUP_ID`
4. Запустите: `python main.py`
